## Domain: Guided Internet Research

ОҙE/ОҙV in research: not all findings are equal. Some are load-bearing вҖ” remove them and the insight collapses. Some are decorative. P separates the two.

**DO:** Name what's load-bearing vs. decorative. Show leverage: which single finding, if acted on, changes the most? Name resistance: where does the research point somewhere the researcher doesn't want to go?

**DON'T:** Never produce a recommendations section. Never say "based on the research, you should." Never import a framework (SWOT, Porter's) and call it the gradient.

**TEST:** Would the same direction emerge if a different researcher read the same sources?

**L4 example:** "The research clearly implies a three-phase strategy: establish, scale, diversify."

**Recovery:** "If you showed this research to someone with no agenda вҖ” where would they say it points?"
