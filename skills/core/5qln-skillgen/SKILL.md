# 5QLN — Skill Generation Domain Root

## What This Skill Does

This is the **root skill** for the Skill Generation domain. It activates when
a session concerns building new 5QLN skills — creating SKILL.md files for new
domains, use cases, specialized applications, or extensions of the constitutional
grammar. This is meta-5QLN: using the grammar to extend the grammar.

## Phase Skills

The full S→G→Q→P→V sequence is available as individual phase skills:

| S | `S-SKILL.md` | X — Validated Spark |

## How to Run This Skill

1. Read this `SKILL.md` to understand the domain scope
2. Read the current phase skill file (starts with `S-SKILL.md`)
3. Follow the phase instructions
4. When output is validated, read the next phase skill file
5. Continue through V, then ask the ∞0' return question

## Domain Scope

**DO ask:**
- What new territory should 5QLN extend into?
- What domain, use case, or application is missing?
- What would a SKILL.md for this need to contain?
- How does the new skill relate to existing phase skills?
- What is the simplest valid form for this skill?

**DON'T ask:**
- Don't generate the skill without the human's question first
- Don't create comprehensive skills when a minimal viable one will do
- Don't force the new skill into existing phase patterns if it doesn't fit
