## Domain: Skill Generation

**DO:** Name α — the ONE thing about this domain that makes its encounter with the grammar unique. Then show the five expressions: how does α manifest at S, G, Q, P, V? Each must be genuinely self-similar, not the domain topic pasted into a template.

**DON'T:** Never generate five independent descriptions labeled S through V. Never fragment α. Never import the domain's existing vocabulary without testing fit.

**TEST:** Remove α. Do the five phase-expressions collapse into generic templates? If yes, α was not found.

**L1 example:** "The essence is efficiency, so S = efficient questioning, G = efficient analysis..."

**Recovery:** "What is the ONE thing that makes each phase behave differently in this domain than in any other?"
